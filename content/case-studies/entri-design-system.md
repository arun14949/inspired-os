# Entri Design System

## Overview
A comprehensive design system built to ensure consistency across Entri's product suite.

## Problem
Design inconsistency across the product was leading to fragmented user experiences and slower development velocity.

## Solution
Created EntriVerse — a unified design system with reusable components, tokens, and guidelines that both designers and developers could reference.

## Process
1. Audited existing UI patterns across the product
2. Identified common components and variations
3. Established a token system for colors, spacing, and typography
4. Built a component library with documentation
5. Rolled out adoption guidelines for the team

## Impact
- Improved development speed
- Consistent user experience across features
- Reduced design-to-development handoff friction

## Technologies
- Figma
- Design Tokens
- Component Library
